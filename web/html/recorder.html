Note: response times will fluctuate based on activity. For example, your second request will be faster than your first if the system was inactive before your request. This is due to the on-demand nature of the instances currently being used. Average response times should be around 2-3 seconds for most requests. The javascript interface (which this page uses) is also slower as it needs to convert the audio from an ogg variable speed container to the format required by the PriVox APIs. In general, using the smaller models STT response times average about 1-2 seconds (depending on input size) and TTS takes about 1-2 seconds to render a typical sentence.
