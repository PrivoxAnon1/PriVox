The Website
***********

.. toctree::
   :maxdepth: 2

The website provides the interface for users to see their production and consumption numbers. It also has a limited 'profile' page which shows the user email address and API key.

.. figure:: _static/profile_top.png
   :class: with-border

   User Profile account info

Users consume resource (technically QSS) each time they request transcription from an API. They also accumulate QSS each time they produce. This information is summarized in the table in the middle of the profile page and it will look something like this ...


.. figure:: _static/profile_middle.png
   :class: with-border

   User Profile consumption overview



Below this table is a list of producer sessions associated with the user. If you have not contributed yet you will see a blank table near the bottom of your page.
